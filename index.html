Test Plot (Sine Wave)

If PyScript works, you should see a sine wave plot below:

import matplotlib.pyplot as plt import numpy as np # Generate sine wave x = np.linspace(0, 10, 100) y = np.sin(x) plt.plot(x, y, color="teal", linewidth=2, marker="o") plt.title("Hello from PyScript + Matplotlib") plt.xlabel("X Axis") plt.ylabel("Y = sin(x)") plt.grid(True, linestyle="--", alpha=0.6) plt.show()
